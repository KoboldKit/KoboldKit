ALChannelSource Class Reference

Inherits from NSObject
Conforms to ALSoundSource
Declared in ALChannelSource.h

Overview

A Sound source composed of other sources. Property values are applied to all sources within the channel. Sounds will get played by any free sources within this channel. If all sources are busy when playback is requested, it will attempt to interrupt a source to free it for playback.
